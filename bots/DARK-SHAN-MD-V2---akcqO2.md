# DARK SHAN MD V2 WhatsApp Bot Documentation

## Overview
DARK SHAN MD V2 is an advanced WhatsApp bot developed by Kushan Sevmina, designed to provide users with a powerful and customizable messaging experience. This bot offers a wide range of features including automated responses, media handling, group management tools, and much more, all wrapped in a sleek dark-themed interface.

## Key Features
- **Multi-functional commands** for entertainment, utilities, and productivity
- **Group management tools** including admin controls and moderation

- **Customizable responses** and automated replies
- **Dark theme interface** for reduced eye strain
- **Secure authentication** to protect user privacy

## Deployment Instructions

### Prerequisites
- Node.js v16 or higher installed
- WhatsApp account with active number
- Basic terminal/command prompt knowledge

### Installation Steps
1. Clone the repository: `git clone [repository-url]`
2. Install dependencies: `npm install`
3. Configure your settings in `config.js`
4. Start the bot: `npm start`
5. Scan the QR code with your WhatsApp mobile app

## Getting Started
After successful deployment:
1. Save the bot's number in your contacts
2. Send `!menu` to see available commands
3. Use `!help [command]` for specific command information
4. Customize settings with admin commands

## Advanced Configuration
For power users:
- Edit `plugins/` folder to add custom features
- Modify `language.json` for localization
- Set up auto-replies in `autoreply.js`
