## About

## Discord TS Bot
Deploy a Discord bot powered by Typescript, discord.js, and Node.js.

##Instructions
To deploy this template, you will need to set up a bot on Discord and obtain its **Client_ID** and **Discord_Token**. Then, deploy the template and you're good to go!
- [How to set up a Discord Bot Application](https://discordjs.guide/preparations/setting-up-a-bot-application.html#creating-your-bot)
- [Find Discord_Token (aka Bot Token)](https://railway.app/template/EWKFBX?referralCode=8A6l-m)
- [Find Client_ID (aka Application ID)](https://support-dev.discord.com/hc/en-us/articles/360028717192-Where-can-I-find-my-Application-Team-Server-ID-#:~:text=enable%20Developer%20Mode.-,APPLICATION%20ID,-The%20Application%20ID)

##How to add your bot to your server
After you've deployed the template, you can add a bot to your server using this [guide](https://discordjs.guide/preparations/adding-your-bot-to-servers.html#creating-and-using-your-invite-link)
- make sure to select ``Send Messages`` and ``Use Slash Commands`` under bot permissions
