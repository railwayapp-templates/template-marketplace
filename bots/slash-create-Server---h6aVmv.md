## About

<div align="center">

<img src="https://raw.githubusercontent.com/Snazzah/slash-create/master/static/textlogo.png" height="50">

/create is a way to create and sync [Discord slash commands](https://discord.com/developers/docs/interactions/application-commands) and serve them over a server.

## What gets deployed

| Service | Source | Type |
|---------|--------|------|
| interactions | [Snazzah/slash-create-template/tree/master](https://github.com/Snazzah/slash-create-template/tree/master) | Web service |

## Environment variables

| Variable | Default | Description |
| --------- | ------- | ----------- |
| `DISCORD_APP_ID` | - | The application ID of the Discord app |
| `DISCORD_BOT_TOKEN` | (secret) | The bot token of the Discord app |
| `DISCORD_PUBLIC_KEY` | - | The public key of the Discord app |
