## About

ibatin-pruebas is a development and testing environment for the IBATIN project. It is used to validate web services, content structures, and digital tools before production release, allowing teams to safely experiment, review changes, and iterate on features in a controlled setup.

Hosting ibatin-pruebas involves deploying a preview and testing version of the IBATIN project on Railway. This includes configuring build and start commands, managing environment variables, and connecting the project repository. Railway handles infrastructure, scaling, and uptime, enabling fast deployments and continuous testing without complex server management.
