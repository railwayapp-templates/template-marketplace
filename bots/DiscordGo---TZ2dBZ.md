## About

## Overview

DiscordGo is a [Go](https://golang.org/) package that provides low level bindings to the [Discord](https://discord.com/) chat client API. 

## Highlights

- Nearly complete support for all of the Discord API endpoints
- Websocket interface
- Voice interface
