## About

### Watch YouTube Walkthrough

[![](https://img.youtube.com/vi/tBTvenDmkdw/0.jpg)](https://youtu.be/tBTvenDmkdw?si=Ztot8vaZdPb4LzyO "How to Make a Drag & Drop 24/7 Discord Bot (Railway, Buildship)")

Drag and drop 24/7 Discord bots (Discord.js) integrated with low-code / no-code backend tools like Buildship, FastGen, Xano or 3rd party serverless endpoints.

Hosted on Railway a 24/7 low-code / no-code instant deployment platform.

Using BuildShip.com you can generate custom code as nodes then drag and drop the custom functionality.

Customize the Discord bot gateway intents as Railway variables.

More detailed instructions: https://github.com/matthewlal/discord.js-bot-nocode/
