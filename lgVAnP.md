## About

Bot Discord polyvalent conçu pour accueillir les nouveaux membres avec style ! Crée des cartes de bienvenue personnalisées avec images, texte et polices uniques. Supporte plusieurs serveurs avec deux instances indépendantes, gestion via tokens séparés et configuration facile avec .env. Parfait pour dynamiser vos communautés !

## Environment variables

| Variable | Default | Description |
| --------- | ------- | ----------- |
| `DATABASE_URL` | - | Mongoose UrI |
| `DATABASE_NAME` | - | Your database name |
| `DISCORD_TOKEN` | (secret) | Discord Token |
