# Deploy wsrv.nl on Railway

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/template/wsrvnl)

## About

[wsrv.nl](https://github.com/weserv/images) (formerly known as weserv) is an image cache &amp; resize service. It resizes your image, cache it worldwide, and display it.

This template allows you to deploy wsrv.nl out-of-the-box with just one-click.

## What gets deployed

| Service | Source | Type |
|---------|--------|------|
| wsrv.nl | `ghcr.io/weserv/images:5.x` | Web service |

## Configuration

- **Networking:** Public domain with automatic HTTPS
