# Deploy PenguinMod storage API on Railway

Host your own server for the PenguinMod storage extension

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/template/seDKg9)

## About

Use this to host your own server for the [PenguinMod](https://penguinmod.com/) storage extension, on [railway.app](https://railway.app/)

 If you don't use a volume, your data will be cleared every time you restart the service  


###Learn more  
[Documentation](https://docs.penguinmod.com/extensions/storage)  
[Source code](https://github.com/TheShovel/PenguinMod-StorageAPI-on-rails)

## What gets deployed

| Service | Source | Type |
|---------|--------|------|
| PenguinMod storage API | [TheShovel/PenguinMod-StorageAPI-on-rails](https://github.com/TheShovel/PenguinMod-StorageAPI-on-rails) | Web service |

## Configuration

- **Networking:** Public domain with automatic HTTPS
- **Volume:** `/app/data`

**Category:** Storage · **Languages:** Go, Shell
