# Deploy Pocketbase with PB Hooks and PocketCMS on Railway

PB with access to Hooks and CMS

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/template/6ckIwL)

## About

This setup provides access to the hooks folder for custom server-side logic and integrates PocketCMS for content management. To access Hooks eject the service from the template so it copies a repo into your Github. Pocket CMS is available ├─ CMS Admin UI: /cms/

## What gets deployed

| Service | Source | Type |
|---------|--------|------|
| pocketbase | [RenlyLtd/pocketbase-JZsk](https://github.com/RenlyLtd/pocketbase-JZsk) | Web service |

## Configuration

- **Networking:** Public domain with automatic HTTPS
- **Volume:** `/root/pocketcms`
