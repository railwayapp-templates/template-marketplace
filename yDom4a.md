# Deploy Next.js to Railway

This is a Next.js template which can be deployed to [Railway](https://railway.app) with zero configuration.

## Deploying to Railway

Railway automatically configures your Next.js application to run as a Node.js server (through `next start`). 

You can deploy to Railway using the [Railway CLI](https://docs.railway.app/guides/cli) or via GitHub.

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/new)

For more information, see our [deployment quickstarts](https://docs.railway.app/guides/languages-frameworks).

## What gets deployed

| Service | Source | Type |
