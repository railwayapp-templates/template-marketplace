## About

den.ooo is a lightweight module registry for Deno to serve JavaScript, TypeScript and WebAssembly files from GitHub, GitLab, NPM and many others.

Our NPM package and this template are licensed under the [MIT license](https://github.com/dendotooo/den.ooo/blob/main/license).

- **Healthcheck:** `/?health`
- **Networking:** Public domain with automatic HTTPS
