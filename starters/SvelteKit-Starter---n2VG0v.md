## About

🧱 SvelteKit Starter Template — Modern & Minimal

A clean and modern starter template for building web applications using SvelteKit. This template is designed with developer productivity in mind — including essential utilities and configurations that help you get started fast.



🚀 Features
	•	✅ SvelteKit (latest version)
	•	🎨 Tailwind CSS – for modern, responsive UI
	•	🌐 Prettier + ESLint – auto-formatting and linting
	•	🔒 dotenv – easy environment variable management
	•	🧩 Lib folder structure – ready for components, stores, utils
	•	🧭 Layout System – +layout.svelte, error page, loading states
	•	🗃️ Type-safe form utils (optional Zod support)
	•	📁 Preconfigured for deployment on Railway
