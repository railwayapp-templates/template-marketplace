## About

TanStack Start Monorepo is a production-ready full-stack starter template combining TanStack Start, Better Auth, Drizzle ORM, and Supabase. It provides file-based routing, type-safe authentication, PostgreSQL database integration, and pre-configured shadcn/ui components with Tailwind CSS v4.

Hosting this template requires a PostgreSQL database and a few environment variables. Railway's one-click PostgreSQL provisioning makes setup straightforward. The included Dockerfile handles the build process, compiling the TanStack Start application with Nitro as the server runtime. After deployment, you'll need to run database migrations once using the Drizzle CLI. Optional Google OAuth requires adding your credentials and updating the authorized redirect URI in Google Cloud Console.

## Environment variables

| Variable | Default | Description |
| --------- | ------- | ----------- |
| `PORT` | 3000 | Same one as in Docker |
| `BASE_URL` | - | Auto-filled by Railway |
| `NODE_ENV` | production | development | production |
| `DATABASE_URL` | postgresql://postgres:postgres@127.0.0.1:54322/postgres | Postgres/Supabase connection string from project settings |
| `BETTER_AUTH_URL` | - | Auto-filled by Railway |
| `TRUSTED_ORIGINS` | - | Auto-filled by Railway |
| `GOOGLE_CLIENT_ID` | not-configured | From https://console.cloud.google.com/apis/credentials |
| `BETTER_AUTH_SECRET` | (secret) | Auto-generated by Railway |
| `GOOGLE_CLIENT_SECRET` | (secret) | Redirect URI: {BETTER_AUTH_URL}/api/auth/callback/google |
