## About

[htmx](https://htmx.org) is a small but mighty libary allowing developers to access modern browser features like AJAX, CSS Transitions, WebSockets, and Server Sent Events directly in HTML. In combination with [Templ](https://templ.guide), a go library, you can build powerful server-side components and modern user interfaces with minimal javascript.
