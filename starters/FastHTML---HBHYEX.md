## About

## FastHTML - Hello World!

This is a super simple FastHTML template to get you going without the necessary set up.

It deploys via Docker using [Poetry](https://python-poetry.org/) as the package manager. You can change this if you want to.

You can expand this as much as you want to as well. Read the [FastHTML docs](https://docs.fastht.ml/) for more information and to get going!

## What gets deployed
