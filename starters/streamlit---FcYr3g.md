## About

A minimal Streamlit app template for Python to build AI/ML applications, Analytics and Data Science tools with ease. One click deployment and development. Rapidly build, deploy, and share beautiful ML and data science web apps. 

No HTML. No writing API endpoints. Just Python.

## What gets deployed

| Service | Source | Type |
|---------|--------|------|
| streamlit | [tkruer/streamlit](https://github.com/tkruer/streamlit) | Web service |

## Configuration

- **Start command:** `python -m streamlit run app.py`
- **Networking:** Public domain with automatic HTTPS
