## About

Starlette is a lightweight ASGI framework/toolkit, which is ideal for building async web services in Python.
This template is based on [github.com/railwayapp-templates/starlette](https://github.com/railwayapp-templates/starlette) and is a template/demo project for using Starlette with Python.

This template shows how to setup Starlette, so you can start creating things with it on your own. Everything is setup once you deploy this template.
