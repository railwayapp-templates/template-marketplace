## About

This is a functional WebSocket server that will get you started with a very solid base around WebSockets that you can adapt to your use case!

Key features:
- WebSockets
  - Lightweight
  - KeepAlive
  - State Map
- TypeScript
  - Environment variables typing
  - TypeScript & JavaScript compatibility
  - ES Modules & CommonJS compatibility
- Node.js
  - Live Reload
  - Debugging
- esbuild
  - Fast bundling
  - Fast transpiling
- Rome
  - Fast linting
  - Fast formatting
  - Import sorting
- GitHub
  - One click template
  - Continuous integration with dependency caching
