## About

## Overview

A Rust Rocket is a web application server framework for the Rust programming language. It is designed to provide an easy-to-use and efficient framework for building fast and reliable web applications.

## Highlights

This deploys a simple web-server with a single route served directly from the `/` route. This starter deploys as quickly in 2 minutes.
