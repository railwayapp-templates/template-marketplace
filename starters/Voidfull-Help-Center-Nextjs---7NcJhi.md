### Setup

To run this template, you need the following **environment variables**.

* `NEXT_PUBLIC_VOIDFULL_SITE_ID`
* `NEXT_PUBLIC_VOIDFULL_CONTENT_TOKEN`

NOTE: Voidfull content token will always start with `voidfull_pat_`

You can find both these variables value by logging into your Voidfull account.

Website: [voidfull.com](https://voidfull.com?utm_source=railway&utm_medium=templates)
Github: https://github.com/voidfullspace

## Environment variables

| Variable | Default | Description |
| --------- | ------- | ----------- |
| `NEXT_PUBLIC_VOIDFULL_SITE_ID` | - | Enter the Site ID |
| `NEXT_PUBLIC_VOIDFULL_CONTENT_TOKEN` | (secret) | Enter the Voidfull Content Token `voidfull_pat_***` |
