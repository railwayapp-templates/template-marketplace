# Deploy Bun WebSockets on Railway

Simple WebSocket Bun server

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/deploy/BLofAq)

## About

A simple WebSocket server to get started using Bun.

[Bun](https://bun.sh) is a:
- Toolkit All-in-one
- JavaScript runtime
- Package manager
- Test runner
- Bundler

You can connect to the template's server using:

```sh
bunx wscat -c wss://bun-ws.up.railway.app
```

Or deploy this template yourself and tinker with it! 🔥

## What gets deployed

| Service | Source | Type |
|---------|--------|------|
| Server | [dayblox/bun-ws](https://github.com/dayblox/bun-ws) | Web service |

## Configuration

- **Start command:** `bun src/index.ts`
- **Networking:** Public domain with automatic HTTPS
