# Deploy Express+GraphQL+Prisma on Railway

Clean template using Express.js, Prisma, GraphQL, SQLite, and Apollo Server

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/deploy/vCAGzF)

## About

This template is a backend service built with Express.js, Prisma ORM, and Apollo Server for GraphQL. Ideal for developers who want to focus on backend functionality while having the option to add their own frontend. If you want to see my frontend solution, check it out here: https://railway.app/template/y1l7kq?referralCode=B4Nuxy
