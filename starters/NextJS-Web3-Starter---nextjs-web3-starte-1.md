# Deploy NextJS Web3 Starter on Railway

This is a Web3 NextJS starter dApp.

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/deploy/nextjs-web3-starte-1)

## About

The app is configured for a simple Railway deployment and showcases some basic smart contract interactions.

## What gets deployed

| Service | Source | Type |
|---------|--------|------|
| nextjs-web3-starter | [I0San/nextjs-web3-starter](https://github.com/I0San/nextjs-web3-starter) | Web service |

## Environment variables

| Variable | Default | Description |
| --------- | ------- | ----------- |
| `NEXT_PUBLIC_APP_URL` | https://reown.com | You application deployed url. Used for Web3 Widget |
| `NEXT_PUBLIC_APP_ICON` | https://reown.com/logo.png | You application icon. Used for Web3 Widget |
| `NEXT_PUBLIC_APP_NAME` | Your App Name | You application name. Used for Web3 Widget |
| `NEXT_PUBLIC_APP_DESCRIPTION` | Your App Description | You application description. Used for Web3 Widget |
| `NEXT_PUBLIC_EXAMPLE_CONTRACT` | 0xf5b1c4f599DFf501f2A622E8809aF7f389394A19 | Demo smart contract on Sepolia |
| `NEXT_PUBLIC_WALLETCONNECT_PROJECT_ID` | - | WalletConnect Project ID Key (Reown) |

## Configuration

- **Healthcheck:** `/health`
- **Networking:** Public domain with automatic HTTPS
