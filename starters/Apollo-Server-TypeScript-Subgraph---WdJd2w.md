# Deploy Apollo Server TypeScript Subgraph on Railway

A federation-compatible TypeScript GraphQL server

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/deploy/WdJd2w)

## About

Quickly add a new subgraph to your federated supergraph with just a couple clicks!

## Features

- Examples for federated entities, including tests
- GitHub Actions which integrate with Apollo GraphOS (detect breaking changes, update your router automatically)
- SDL-first developer experience leveraging [graphql-codegen](https://the-guild.dev/graphql/codegen)
- Maintained by Apollo

## What gets deployed

| Service | Source | Type |
|---------|--------|------|
| subgraph | [apollographql/subgraph-template-typescript-apollo-server-boilerplate](https://github.com/apollographql/subgraph-template-typescript-apollo-server-boilerplate) | Web service |
