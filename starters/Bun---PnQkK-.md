## About

### Overview
[Bun](https://bun.sh) is an all-in-one JavaScript runtime & toolkit designed for speed, complete with a bundler, test runner, and Node.js-compatible package manager.

Using the provided Dockerfile, this template deploys a minimal web application based on the Bun runtime. 

### Learn More
* Bun web application [GitHub repo](https://github.com/alphasecio/bun/blob/main/README.md)
