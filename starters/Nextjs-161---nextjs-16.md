# Deploy Next.js 16.1 on Railway

Next.js 16.1 bundled with TailwindCSS 4 and Shadcn UI

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/template/nextjs-16)

## About

[What is Next.js 16.1? This template is a standard [NextJS 16.1](https://nextjs.org/blog/next-16-1) deployment. It's configured to use [TailwindCSS 4](https://www.tailwindcss.com) out of the box, and is configured with [shadcn UI](https://ui.shadcn.com) for stylinigs. 

It's meant to be a building block for quick projects using the newest NextJS features.

This application deploys fine onto railway a single deployment and will run Next.js client and server side functionality. Nothing overly special!
