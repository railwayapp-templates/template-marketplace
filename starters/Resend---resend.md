# Deploy Resend on Railway

Send transactional and marketing emails from your app

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/deploy/resend)

## About

Resend is an email platform for developers that enables robust transactional flows and polished marketing campaigns.

Hosting Resend gives you the power to send emails from your application with full observability, idempotency (duplicate sending protection), and more. You can also add full marketing flows including audience management, an interactive editor, and comprehensive analytics.

## What gets deployed

| Service | Source | Type |
|---------|--------|------|
| Resend Starter | [resend/resend-node-railway-starter](https://github.com/resend/resend-node-railway-starter) | Web service |

## Environment variables

| Variable | Default | Description |
| --------- | ------- | ----------- |
| `RESEND_API_KEY` | (secret) | API Key from your Resend account. You can create a new key at https://resend.com/api-keys |
