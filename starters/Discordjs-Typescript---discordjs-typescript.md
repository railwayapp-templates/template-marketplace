# Deploy Discord.js Typescript on Railway

Discord.js Typescript starter template. Create a fork to modify.

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/deploy/discordjs-typescript)

## About

discord.js is a powerful [Node.js](https://nodejs.org) module that allows you to easily interact with the [Discord API](https://discord.com/developers/docs/intro).

In order to modify the code in this template, you'll need to fork the root repository and make your changes: https://github.com/thesamgordon/railway-discord.js-typescript

The `TOKEN` variable should be set to your Discord bot's token. Instructions for creating one can be found [here](https://discordgsm.com/guide/how-to-get-a-discord-bot-token).

## What gets deployed

| Service | Source | Type |
|---------|--------|------|
| Discord.js Template | [thesamgordon/railway-discord.js-typescript](https://github.com/thesamgordon/railway-discord.js-typescript) | Worker |

## Environment variables

| Variable | Default | Description |
| --------- | ------- | ----------- |
| `TOKEN` | (secret) | Your Discord bot's token. Create a bot at https://discord.com/developers/applications. |
