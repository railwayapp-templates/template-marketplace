## About

This starter project contains an empty Cobol project that takes full advantage of the nixpacks GNU Cobol  support nixpacks. To get started simply update the main cobol.

For more information on gnu cobol:
https://gnucobol.sourceforge.io/

For more information on how to configure your build:
https://nixpacks.com/docs/providers/cobol
