# Deploy 1.20.1 Pumpkin Minecraft Server on Railway

A ready-to-use 1.20.1 Minecraft server running the Pumpkin server software.

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/template/1201-pumpkin-minec-1)

## About

This template lets you deploy a **Minecraft Java Edition 1.20.1 Pumpkin server** on Railway with a single click. Pumpkin is a modern Minecraft server focused on performance and simplicity.
⚠️ **Pumpkin is still in active development**, so some features may be missing or not fully working yet. This makes it best for testing, experimenting, and small private servers.

With this template, hosting a Pumpkin Minecraft server is effortless. Railway handles the setup, hosting, and restarts automatically, so you don’t need to configure servers or manage infrastructure yourself. After deployment, you simply join the server using the address provided by Railway. This template is ideal if you want to quickly spin up a server to try Pumpkin, play with friends, or explore an alternative Minecraft server implementation without any manual setup.
