# Deploy Remix on Railway

Starter Remix JS app using create-remix@2.9.2

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/template/4nlQ1-)

## About

This is a basic Remix app that can be deployed with one click and built upon.

The nixpacks file has been updated here to explicitly omit dev-dependencies on the server using --omit=dev rather than relying on an environment variable that specifies production mode. No warnings or errors show up at build/deploy stage.

Make sure you have a url setup so you can access your new Remix app!

## What gets deployed

| Service | Source | Type |
|---------|--------|------|
| railway-remix | [haydenlingle/railway-remix](https://github.com/haydenlingle/railway-remix) | Web service |
