# Deploy Pokemon Showdown Server on Railway

Deploy your own Pokemon Showdown battle server

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/template/pokemon-showdown)

## About

**One-click deployment of Pokemon Showdown server with:**

- Pre-configured admin account setup for immediate moderation control
- Docker-based reliable deployment with automatic scaling
- Automatic SSL certificates and custom domain support
- Ready-to-use battle simulator and team builder for all Pokemon generations
- Built-in chat system and community features for your players

## What gets deployed

| Service | Source | Type |
|---------|--------|------|
| Pokemon Showdown | [smolpaw/pokemon-showdown-railway](https://github.com/smolpaw/pokemon-showdown-railway) (root: /) | Web service |

## Environment variables

| Variable | Default | Description |
| --------- | ------- | ----------- |
| `ADMIN_USER` | (secret) | User name for Administrator account |
