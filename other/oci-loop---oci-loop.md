## About

oracle-loop (OCI Always Free Provisioner) is a web-based automation tool designed to continuously attempt to provision Oracle Cloud Infrastructure (OCI) Always Free resources,

Hosting oracle-loop involves deploying a Python Flask application that communicates directly with the Oracle Cloud Infrastructure (OCI) API.
