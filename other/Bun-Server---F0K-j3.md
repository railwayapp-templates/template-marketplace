# Deploy Bun Server on Railway

A simple server using Bun

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/deploy/F0K-j3)

## About

[Bun](https://bun.sh/) is a new JavaScript runtime that is fast and is a drop-in replacement for NodeJS. This template uses Nixpacks to deploy on Railway.

The server itself just follows the getting started guide on bun.sh. Easy as pie! Deploy to quickly spin up a repo that uses Bun. Or simply browse the repo and copy the code.

## What gets deployed

| Service | Source | Type |
|---------|--------|------|
| server | [coffee-cup/railway-bun-init](https://github.com/coffee-cup/railway-bun-init) | Web service |
