# Deploy Rust starter on Railway

A simple web-server written in Rust using axum

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/deploy/5HAMxu)

## About

This template provides a simple web-server, featuring the key concepts of `axum` and `tokio`.

This template has lots of comments to walk you through how everything works, and how it can be modified.

This template uses `axum`s Router, to create an app router, to which routes are added. A server is then initialised, which starts listening for requests. These route functions talk about the different way a route function can work, ie, some returning strings, and some providing much more complex responses.
