## About

Resume Builder is a free, open-source, ATS-friendly resume creator built with Next.js that runs entirely in the browser. It helps users create professional resumes using a structured document model while offering customizable visual themes. Resume data is stored locally in the browser, with no accounts or server-side storage required.

Hosting Resume Builder on Railway provides a fast and reliable way to deploy the application without managing infrastructure. Railway automatically builds and deploys the project from your repository, provisions HTTPS, and makes it easy to connect a custom domain. Since resume data is stored locally in each user's browser using IndexedDB, the application does not require a backend database or persistent server storage. Railway handles networking, deployments, and scaling, allowing you to focus on maintaining the application while providing users with a responsive, privacy-first resume builder that works online and continues functioning offline after the initial load.

## Environment variables

| Variable | Default | Description |
| --------- | ------- | ----------- |
| `APP_URL` | - | Public URL generated by Railway — set at runtime by Railway: |
| `NODE_ENV` | production | Node environment (required to ensure Next runs in production on Railway) |
| `SECRET_KEY` | (secret) | Optional: If you want a server-side secret for any features |
| `NEXT_PUBLIC_TURNSTILE_SITE_KEY` | 1 | Public Turnstile site key used client-side (inlined at build time). Leave empty if you don't use Turnstile. |
