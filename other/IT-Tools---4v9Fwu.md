## About

A collection of online developer tools, providing a rich suite of tools for developers to use anytime, anywhere.

IT-Tools is an open-source collection of handy, browser-based tools for developers and IT professionals. Built with Vue.js and TypeScript, it includes utilities like UUID generators, JWT decoders, format converters, and more. The project focuses on providing a great user experience and can be easily self-hosted via Docker, Cloudron, Tipi, or Unraid. Licensed under GNU GPLv3, it is actively maintained and welcomes community contributions to continuously expand its features.
