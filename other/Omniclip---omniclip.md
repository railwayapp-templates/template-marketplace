## About

Omniclip is a free and open source video editing application that runs entirely in the browser. It provides powerful editing tools for video, audio, images, and text while keeping projects private by processing everything locally. Developers can also integrate Omniclip's reusable web components into their own applications to build custom video editing experiences.

Hosting Omniclip on Railway makes it easy to deploy the web application on reliable cloud infrastructure without managing servers manually. Railway automatically builds the project from your repository, provisions HTTPS, and provides public networking so users can securely access the application from anywhere. Since media processing and project storage are designed to happen locally in the user's browser, the application does not require server-side storage or a database for standard deployments. Railway also simplifies continuous deployments, environment variable management, and scaling, allowing you to focus on developing new features while Railway manages the underlying infrastructure.
