## About

Toybox is a BSD-licensed collection of Linux command-line utilities in a single binary. Used as the default tools in Android, it provides over 200 commands like ls, grep, find, and sed for lightweight debugging.

Deploy Toybox on Railway for a minimal shell environment accessible via SSH. Useful for debugging, network testing, and running scripts between services.
