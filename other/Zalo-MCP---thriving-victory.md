## About

Zalo Personal Server is a lightweight Node.js backend that connects your personal Zalo account via QR code. It provides REST APIs for messaging, contacts, and conversations — designed to work with AI Assistant Chrome extensions.

Deploying thriving-victory on Railway gives you a private, always-on Zalo backend. The server handles QR-based authentication, session persistence across restarts, and exposes simple REST endpoints. Once deployed, paste your Railway URL into the Chrome extension settings to start sending and receiving Zalo messages through AI. No coding or configuration required — just deploy, copy your URL, and connect.
