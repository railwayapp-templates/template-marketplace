# Deploy MindsDB on Railway

MindsDB is an open-source platform brings AI directly to databases.

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/template/mindsdb)

## About

MindsDB is an open-source AI platform that enables developers to build, train, and deploy machine learning models directly inside databases using SQL. It simplifies adding predictive and generative AI capabilities to applications without complex ML pipelines.

Hosting MindsDB involves running its server, configuring database connections, and managing integrations with AI models and external data sources. When deployed on Railway, MindsDB runs as a managed service with automatic builds, environment variable management, and scalable infrastructure. You typically configure storage, connect supported databases (like PostgreSQL or MySQL), and expose the API or SQL interface. Railway removes the need for manual server setup, making it easy to deploy, update, and scale MindsDB as your application grows.
