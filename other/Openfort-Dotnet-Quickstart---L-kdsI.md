# Deploy Openfort Dotnet Quickstart on Railway

Dotnet backend quickstart using Openfort to interact with smart wallets.

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/template/L-kdsI)

## About

Openfort is a suite of APIs powering blockchain account interactions for games of all sizes. 

With Openfort, you can interact smart accounts that offer the latest capabilities on account abstraction like:
- Gasless blockchain transactions.
- Payment of gas in ERC-20 tokens.
- Transfer ownership of a wallet without exposing the private key.
- Recovering a smart wallet when access has been lost

With just one click, deploy your Dotnet backend API to the Railway cloud platform.

## What gets deployed

| Service | Source | Type |
