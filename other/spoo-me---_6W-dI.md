# Deploy spoo-me on Railway

Self-Host spoo.me URL shortener on railway.app

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/template/_6W-dI)

## About

spoo.me is a free, open-source service for shortening URLs. It offers URL statistics, a free API, and customization options. You can create custom slugs, add password protection, and manage link lifespans.

Deploying steps:

- Create a mongoDB server
- Create 2 webhooks
- Click on the deploy now button of this template
- Fill in the required environment variables
- Wait for the deployment to finish

## What gets deployed

| Service | Source | Type |
|---------|--------|------|
| url-shortener | [spoo-me/url-shortener](https://github.com/spoo-me/url-shortener) | Web service |
