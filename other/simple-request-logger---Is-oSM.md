# Deploy simple-request-logger on Railway

Simple request logger for debugging webhooks that require public urls.

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/template/Is-oSM)

## About

I made this as a simple request logger to debug webhooks. Just throw up a deployment in railway.app and watch the logs. Written in nodejs + express.

Steps:
1. Deploy template to railway
2. Point webhooks to railway provider URL
3. Open deployment logs
4. Trigger Webhook

## What gets deployed

| Service | Source | Type |
|---------|--------|------|
| request-logger | [foopis23/request-logger](https://github.com/foopis23/request-logger) | Web service |

## Configuration

- **Networking:** Public domain with automatic HTTPS
