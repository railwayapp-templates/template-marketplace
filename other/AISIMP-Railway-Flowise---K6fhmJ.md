# Deploy AISIMP-Railway-Flowise on Railway

Deploying Flowise with Password and Username

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/deploy/K6fhmJ)

## About

This Template will deploy Flowise with two variables. This template will be updated with the latest repo of Flowise. 

Make sure after deployment to change your usernam and password. Default values are


FLOWISE_USERNAME=Aisimp
FLOWISE_PASSWORD=BuildingBots


you can then easily change them in the Flowise environment

## What gets deployed

| Service | Source | Type |
|---------|--------|------|
| Canybec Sulayman | [canybec/AISIMP-Railway-Flowise](https://github.com/canybec/AISIMP-Railway-Flowise) | Web service |

## Environment variables

| Variable | Default |
| --------- | ------- |
| `FLOWISE_PASSWORD` | (secret) |
| `FLOWISE_USERNAME` | (secret) |
