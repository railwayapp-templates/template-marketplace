# Deploy stirling_pdf_improved on Railway

Powerfull PDF editor, with persistent passwords and ALL languages for OCR

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/template/EZDbH1)

## About

With this template you do have a login authentification by default :

Default login /password : admin / stirling

Please make sure to change it after deploy !

But you also have ALL languages for OCR recognition integrated (not just English)

With the persistent storage, you will not loose your created accounts on redeploy.

Enjoy ! :)
