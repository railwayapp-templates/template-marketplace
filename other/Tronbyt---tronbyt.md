# Deploy Tronbyt on Railway

An ecosystem replacement for Tidbyt

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/deploy/tronbyt)

## About

Tronbyt is an ecosystem replacement for Tidbyt.

Tidbyt was [acquired by Modal](https://modal.com/blog/tidbyt-is-joining-modal) in Nov 2024 and there have been no updates since. The apps that still exist also don't work well anymore so Tronbyt helps fix that by allowing you to host your own server and also create your own apps if you wish to do so.

Hosting with this template is straightforward and should take a single click.

By default, you will be prompted to create an admin user after your first deploy and should be able to use those details to log in in the future.

## What gets deployed

| Service | Source | Type |
|---------|--------|------|
| tronbyt/server | `ghcr.io/tronbyt/server` | Web service |
