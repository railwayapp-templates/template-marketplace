## About

Kali Linux is a Debian-based distribution designed for penetration testing and security auditing. This template provides a persistent Kali environment with 600+ pre-installed security tools on Railway.

Deploying Kali Linux on Railway gives you a persistent security-focused Linux environment accessible via SSH. This is useful for running security assessments, penetration testing, network reconnaissance, and having a full suite of security tools available on demand. The container stays online and can be accessed anytime using Railway's SSH command feature.
