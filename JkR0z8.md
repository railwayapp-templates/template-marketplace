# Deploy Lago (working) on Railway

Open-source software for metering and usage-based billing.

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/template/JkR0z8)

## About

Original Lago template made by another user was not working so I made this one.
It will spin a postgres db, a redis db and both the frontend and api images necessary to run lago in the simplest manner.
After deployed, you can access the service through the frontend url (after creating an account).

## What gets deployed

| Service | Source | Type |
|---------|--------|------|
| lago-api | `getlago/api:v0.49.1-beta` | Web service |
| Postgres | `ghcr.io/railwayapp-templates/postgres-ssl:latest` | Database |
| Redis | `bitnami/redis` | Database |
| lago-front | `getlago/front:v0.49.1-beta` | Web service |

## Configuration

- **Start command:** `./scripts/start.sh`
- **Networking:** Public domain with automatic HTTPS
- **Volume:** `/app/storage`
- **TCP Proxies:** 5432
- **Volume:** `/var/lib/postgresql/data`
- **TCP Proxies:** 6379
- **Volume:** `/bitnami`
