## About

## Chroma - the AI-native open-source embedding database. 

It has all the tools you need to use embeddings.

This template allows you to spin-up a chroma DB in just a few clicks.

Uses official chroma docker image and mounts a volume, you can talk to chroma on internal networking port 8000.
