# Deploy worldmonitor on Railway

Real-time global intelligence dashboard with interactive map

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/deploy/worldmonitor)

## About

WorldMonitor runs on Railway as a single Docker-based web service, delivering a browser-ready global intelligence dashboard with no source build steps.

## What gets deployed

| Service | Source | Type |
|---------|--------|------|
| app | `xiaosong233/worldmonitor-railway:latest` | Web service |

## Environment variables

| Variable | Default |
| --------- | ------- |
| `PORT` | 80 |
| `API_UPSTREAM` | https://api.worldmonitor.app |
