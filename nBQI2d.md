# Deploy Authentik on Railway

Open-source Identity Provider that emphasizes flexibility and versatility.

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/template/nBQI2d)

## About

Authentik can be seamlessly integrated into existing environments to support new protocols. authentik is also a great solution for implementing sign-up, recovery, and other similar features in your application, saving you the hassle of dealing with them.

This template deploys the required docker images for it to work.
For initial configuration, visit your server url at {URL}/if/flow/initial-setup/

## What gets deployed

| Service | Source | Type |
|---------|--------|------|
| Redis | `bitnami/redis` | Database |
| Postgres | `ghcr.io/railwayapp-templates/postgres-ssl:latest` | Database |
| server | [leonardochappuis/authentik-railway](https://github.com/leonardochappuis/authentik-railway) | Web service |
| worker | [leonardochappuis/authentik-railway](https://github.com/leonardochappuis/authentik-railway) (root: /worker) | Worker |
