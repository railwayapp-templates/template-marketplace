# Deploy EDA on Railway

Web application for data analysis and visualization.

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/template/eda)

## About

Web application for data analysis and visualization.

All you need to get started is in the provided image. Once deployed, enable networking, then login using the default credentials and connect many different kinds of databases.

### Default credentials:
Username: `eda@jortilles.com`
Password: `default`

## What gets deployed

| Service | Source | Type |
|---------|--------|------|
| eda | `jortilles/eda:latest` | Web service |

## Configuration

- **Networking:** Public domain with automatic HTTPS

**Category:** Analytics
