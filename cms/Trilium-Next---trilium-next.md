## About

Trilium Next is a free and open-source, cross-platform hierarchical note taking application with focus on building large personal knowledge bases. 

TriliumNext started as a fork of the original Trilium repository and provides markdown, image, to-do lists, and table support, along with interlinking, mind maps, flowcharts, and code blocks for comprehensive knowledge management.

Trilium is a self-hostable note taking application, with a heavy focus on customizability and security. Setting up this template requires no configuration and can be deployed in a few clicks.
