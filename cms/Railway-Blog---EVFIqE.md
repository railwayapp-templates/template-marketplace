## About

## Overview

Deploy the official [Railway blog repo](https://developers.notion.com/), which uses the [Notion API](https://developers.notion.com/) to convert pages into blog posts.

## Highlights

- NextJS
- TypeScript
- TailwindCSS
- Notion

## Learn More

View the production deployment of official **Railway Blog** to see an example.

## What gets deployed

| Service | Source | Type |
|---------|--------|------|
| NextJS Backend | [railwayapp/blog](https://github.com/railwayapp/blog) | Web service |

## Environment variables

| Variable | Default | Description |
| --------- | ------- | ----------- |
| `POSTS_TABLE_ID` | - | The database ID for the table containing your posts. |
| `NOTION_API_TOKEN` | (secret) | The token for your Notion integration. |
