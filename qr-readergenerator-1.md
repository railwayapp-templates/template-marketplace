## About

QR Reader/Generator API is a FastAPI-based application that allows users to generate QR codes from text or URLs and decode QR codes from image uploads. It's lightweight, fast, and ideal for integrating into web apps, authentication systems, and product tools.

Hosting the QR Reader/Generator API on Railway means effortless deployment and scaling. Railway provides the infrastructure-as-a-service layer, so developers can focus purely on writing application logic. The API exposes endpoints to generate QR codes in PNG format and decode them using image uploads. With Railway, there's no need to manage Docker, servers, or CI/CD manually—just link your GitHub repository, and Railway handles the rest. The platform is built for speed, simplicity, and scalability, making it perfect for small tools or production-grade services.
